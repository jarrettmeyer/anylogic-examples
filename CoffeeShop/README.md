# CoffeeShop

## Parameters

| Name                     | Type    | Description                                                              |
| :----------------------- | :------ | :----------------------------------------------------------------------- |
| dailyOpenTime            | double  | What time of day (in minutes) does the coffee shop open?                 |
| dailyCloseTime           | double  | What time of day (in minutes) does the coffee shop close?                |
| minTimeToFillOrder       | double  | Minimum time to fill order.                                              |
| modeTimeToFillOrder      | double  | Mode time to fill order.                                                 |
| maxTimeToFillOrder       | double  | Maximum time to fill order.                                              |
| reviewsAffectArrivalRate | boolean | This parameter is used as a feature flag for arrival rate modifications. |
