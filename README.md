# AnyLogic Examples

Reference examples for [AnyLogic](https://www.anylogic.com/) simulation modeling, designed to illustrate common patterns and design decisions. All examples are built with AnyLogic Personal Learning Edition v8.9.8.

## Examples

- [CoffeeShop](./CoffeeShop/README.md) — Discrete-event simulation of a coffee shop with customer feedback loops and dynamic worker hiring.
- [MachineDegradation](./MachineDegradation/README.md) — Agent-based simulation of a machine fleet with coupled operation and condition statecharts.
